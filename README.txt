stock_dashboard
==========
Python program for visualising COVID data maintained by John's Hopkins.
Updated data can be found here:  https://github.com/CSSEGISandData/COVID-19/tree/master/csse_covid_19_data/csse_covid_19_time_series

The intent of this program is for personal use.


Getting Started
---------------

- Change directory into your newly created project.

    cd covid-19_us

- Create a Python virtual environment.

    python3 -m venv venv

- Upgrade packaging tools.

    env/bin/pip install --upgrade pip setuptools
